# Delinea PAM TSS Client

Minimal cross-platform Python script to retrieve a secret password from **Delinea Secret Server (on-prem)** using SDK Client Rule onboarding authentication.

## Prerequisites

- Python 3.8+
- Network access to your Secret Server instance
- An **SDK Client Rule** created in Secret Server with the "Require Onboarding Key" option enabled.

## Setup

```bash
# Enter the repo
cd pam-tss-client

# (Optional) Create a virtual environment
python -m venv .venv
.venv\Scripts\activate   # Windows
# source .venv/bin/activate  # Linux/macOS

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your actual values
```

## Configuration

| Variable | Description | Example |
|---|---|---|
| `TSS_BASE_URL` | Base URL of your Secret Server | `https://ss.corp.local/SecretServer` |
| `TSS_CLIENT_NAME` | Name of the SDK Client Onboarding Rule | `ProductionWebApp` |
| `TSS_ONBOARDING_KEY` | The onboarding key generated by the rule | `ab12...xyz` |
| `SECRET_ID` | Numeric ID of the secret to retrieve | `42` |

## Run

On every execution, the script will:
1. Ensure the client is onboarded (by calling the REST API and saving credentials locally inside `.tss_client_creds`).
2. Fetch an OAuth2 token using the onboarded credentials.
3. Retrieve and display the requested secret.

```bash
python src/client.py
```

**Output:**
```
Retrieved username: adminUser
Retrieved password: s3cretP@ss!
```
